================================================================================
TEKNISK ARKITEKTUR
================================================================================

PRIMÄR METOD (för alla 5 sajter):
  playwright_get(url) - Simulerar riktig webbläsare, väntar på JS-rendering
  └─ Kringgår bot-detection (403 Forbidden)
  └─ Väntar på networkidle innan HTML-extraction
  └─ Använder headless Chromium-instans

SEKUNDÄRT PARSNINGSFORMAT (för varje sajt):
  1. from_text() - Regex för "24K 1234 kr/g" eller "24 karat 1234 kr per gram"
  2. from_table() - Parsar HTML-tabeller med <tr><td> struktur
  3. Custom regex - Sajt-specifika mönster för variabla format

FALLBACK-NIVÅER:
  1. Playwright + from_text()
  2. Playwright + from_table()
  3. Requests + all parsing
  4. Requests + regex
